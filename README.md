\# CrashGuard AI - Emergency Response System



CrashGuard AI is an AI-based emergency response system that detects accident scenarios, triggers SOS alerts with live location, notifies emergency contacts, and provides a hospital dashboard for faster emergency response.



\## Features



\- Accident detection simulation

\- Driving mode monitoring

\- Voice warning and countdown

\- SOS alert generation

\- Live location sharing

\- Emergency contact notification

\- Hospital dashboard

\- Firebase-based alert storage

\- Telegram alert integration



\## Tech Stack



\- Frontend: React.js

\- Backend: Node.js, Express.js

\- Database: Firebase Realtime Database

\- Alerts: Telegram Bot API

\- Maps: Google Maps location link

\- Tools: Git, GitHub, VS Code



\## Project Purpose



The main goal of CrashGuard AI is to reduce emergency response time after accidents by automatically sending alerts with live location details to emergency contacts and hospitals.
